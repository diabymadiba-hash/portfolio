Kasa — Plateforme de location immobilière

Développement front-end complet d'une application de location entre particuliers avec React : composants réutilisables, routing, galerie d'images et accordéons animés.

Voir le site live GitHub

Contexte du projet

Kasa est une entreprise leader de la location d'appartements entre particuliers en France. Le projet consiste à développer entièrement le front-end de leur nouvelle plateforme web à partir de maquettes Figma, en utilisant React et React Router pour créer une Single Page Application moderne et performante.

Objectifs

🎯Développer une SPA avec React et React Router v6
🎯Créer des composants réutilisables (Slideshow, Collapse, Card)
🎯Gérer la navigation et les routes dynamiques
🎯Implémenter une galerie d'images avec navigation
🎯Créer des accordéons avec animations CSS
🎯Respecter les maquettes Figma pixel-perfect

Compétences développées

✓Architecture de composants React (props, state, hooks)
✓React Router v6 : routes imbriquées, routes dynamiques, 404
✓Sass : nesting, variables, mixins, structure BEM
✓Composant Slideshow avec useState et navigation circulaire
✓Composant Collapse (accordéon) animé
✓Chargement des données depuis un fichier JSON local

Résultats

Application entièrement responsive déployée sur Vercel. La navigation entre les pages est instantanée grâce au routing côté client. Les composants Slideshow et Collapse sont entièrement fonctionnels avec des animations fluides. L'architecture de composants est propre et maintenable, avec une séparation claire des responsabilités.
