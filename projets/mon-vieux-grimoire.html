Mon Vieux Grimoire — Back-end de notation de livres

Développement complet du back-end d'un site de référencement et notation de livres : API REST sécurisée, authentification JWT, upload d'images optimisées et Green Code.

Voir l'API live GitHub

Contexte du projet

Mon Vieux Grimoire est une chaîne de librairies souhaitant ouvrir une plateforme de référencement et notation de livres. Un front-end React était déjà fourni. Ma mission : développer entièrement le back-end en Node.js/Express, concevoir la base de données MongoDB et brancher l'API sur le front-end existant en respectant les spécifications techniques et les principes du Green Code.

Objectifs

🎯Créer un serveur Express avec architecture MVC
🎯Modéliser les données avec Mongoose (User, Book)
🎯Implémenter l'authentification sécurisée JWT
🎯Gérer l'upload et l'optimisation des images (Sharp)
🎯Respecter les principes du Green Code (images WebP légères)
🎯Brancher le back-end sur le front-end React fourni

Compétences développées

✓Architecture MVC avec Node.js et Express
✓Modélisation MongoDB avec Mongoose (schémas, validations)
✓Authentification : hachage bcrypt + tokens JWT
✓Upload de fichiers avec Multer + optimisation Sharp (WebP)
✓Middlewares : authentification, gestion des erreurs, CORS
✓Green Code : réduction du poids des images de 80%

Résultats

Le back-end est entièrement fonctionnel et branché sur le front-end React. Les routes CRUD pour les livres, l'authentification et le système de notation sont opérationnels. L'optimisation des images via Sharp réduit leur poids de plus de 80% (Green Code). L'API est déployée sur Vercel et reste accessible 24h/24.
