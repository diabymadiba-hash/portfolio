Nina Carducci — Optimisation SEO & Débogage

Audit, débogage et optimisation SEO complète du site portfolio d'une photographe : amélioration des performances, accessibilité, référencement naturel et données structurées Schema.org.

Voir le site optimisé GitHub

Contexte du projet

Nina Carducci est photographe professionnelle. Son site existait mais souffrait de problèmes de performance, de bugs JavaScript et d'un mauvais référencement. Ma mission : auditer le site avec Chrome DevTools et Lighthouse, corriger les bugs, optimiser les images, améliorer l'accessibilité et mettre en place une stratégie SEO on-page complète.

Objectifs

🎯Auditer les performances avec Lighthouse et Wave
🎯Corriger les bugs JavaScript identifiés
🎯Optimiser et compresser les images (WebP, lazy loading)
🎯Améliorer l'accessibilité (WCAG 2.1)
🎯Ajouter les balises méta SEO et Open Graph
🎯Implémenter les données structurées Schema.org

Compétences développées

✓Audit Lighthouse (Performance, SEO, Accessibilité, Best Practices)
✓Débogage JavaScript avec Chrome DevTools
✓Optimisation d'images : formats modernes, compression, lazy loading
✓SEO on-page : balises title, meta description, balises Hn, attributs alt
✓Données structurées JSON-LD (LocalBusiness, Schema.org)
✓Accessibilité : contrastes, rôles ARIA, navigation clavier

Résultats

Score Lighthouse Performance : de 52 → 92/100. Score SEO : de 48 → 100/100. Score Accessibilité : de 71 → 97/100. Le poids total des images a été réduit de 60%. La page s'affiche désormais en moins d'1,5 seconde contre plus de 5 secondes initialement. Un rapport d'audit complet avant/après a été livré au client.
