Booki — Page d'accueil d'une agence de voyage

Intégration complète de la page d'accueil d'une agence de voyage fictive à partir de maquettes Figma, en HTML et CSS pur, entièrement responsive.

Voir le site live GitHub
✈️

Contexte du projet

Booki est le premier projet de la formation Développeur Web d'OpenClassrooms. L'objectif était d'intégrer la page d'accueil d'une start-up fictive proposant la réservation d'hébergements et d'activités touristiques, à partir de maquettes Figma fournies par une équipe de designers. Aucun framework CSS n'était autorisé : uniquement HTML5 et CSS3.

Objectifs

🎯Intégrer une maquette Figma pixel-perfect
🎯Créer une mise en page responsive (mobile, tablette, desktop)
🎯Utiliser Flexbox pour les agencements complexes
🎯Respecter les standards W3C (HTML valide, CSS valide)
🎯Gérer les media queries pour 3 breakpoints
🎯Structurer le code de façon claire et maintenable

Compétences développées

✓Maîtrise de la sémantique HTML5 (header, nav, main, section, article, footer)
✓Flexbox avancé : direction, wrap, align, justify, gap
✓Responsive design : media queries à 768px et 1024px
✓Variables CSS et organisation du code
✓Lecture et interprétation de maquettes Figma
✓Validation du code via les outils W3C

Résultats

La page a été intégrée avec fidélité aux maquettes sur les trois formats d'écran. Le code HTML est validé W3C sans erreur. Le site est déployé sur GitHub Pages et entièrement fonctionnel. Ce projet a posé les bases solides de l'intégration web que j'ai appliquées sur tous les projets suivants.
